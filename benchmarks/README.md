# Benchmarks

These are the benchmarks used for our paper

## QF_FP

QF_FP originates from SMTLIB2. You need to execute the command `tar -xzvf QF_FP.tar.gz` to extract it.

## QF_program

QF_FP originates from real-world floating-point program. You need to execute the command `tar -xzvf QF_program.tar.gz` to extract it.
